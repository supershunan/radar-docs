# 预警策略

> 预警策略页面用于配置子项目的预警规则，包括预警算法类型、预警等级、预警点配置等功能。

![](/../vpstatic/images/预警策略介绍.png)

## 1. 预警策略基本信息

- **子项目名称**：显示该预警策略所关联的子项目名称。
- **预警设置状态**：当前预警策略的启用状态，分为“启用”和“禁用”。
- **预警类型**：触发预警的类型，如曲线预警或图像预警。
- **面积阈值**（单位：平方米）：当形变面积超过设定值时触发预警。
- **预警间隔**（单位：小时）：在设置的间隔时间内，同一视图预警区域内、同一级别的预警不会重复触发（仅适用于 2-4 级）。若本次预警等级提升，系统将立即触发新的预警，并从该时刻重新计时。
- **最高等级预警值差**（单位：mm）：触发新的 1 级预警时，需要当前 1 级预警值与上一次 1 级预警值的差值超过该字段所设置的阈值。
- **预警条件**：预警触发的具体条件描述，分为“位移”和“速度”。

## 2. 编辑预警策略

- **编辑**：修改预警策略的各项参数配置。
- **启用**：开启或关闭该预警策略。
- **配置等级**：设置不同预警等级（如蓝色、黄色、橙色、红色）的阈值范围，也可以设置正向和负向。
- **预警点配置**：指定该预警策略应用于哪些监测点或监测面。
