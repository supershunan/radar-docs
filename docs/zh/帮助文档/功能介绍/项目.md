# 项目

> 项目页面用于管理所有监测项目，支持项目的新建、编辑、删除、子项目管理及相关配置。

![](/../vpstatic/images/项目介绍.png)

## 1. 项目基本信息

- **编号**：项目的唯一标识，系统自动生成。
- **项目名称**：自定义的项目名称，用于区分不同项目。
- **经度**：项目的经度坐标。
- **纬度**：项目的纬度坐标。
- **项目地址**：项目的详细地理位置。
- **创建时间**：项目的创建日期，由系统自动记录。

## 2. 编辑项目

- **新增**：点击可创建新的项目。
- **编辑**：修改当前项目的名称、地址、经纬度等信息。
- **删除**：删除当前项目（若当前项目下存在子项目，需要先删除子项目）。
- **添加子项目**：在当前项目下快速新建子项目。
- **数据联网**：配置项目与外部平台的联网接入。
- **联网数据推送记录**：查看已联网数据的历史推送记录及网络连接状态。
- **第三方传感器接入**：配置非本系统的第三方传感器数据接入。
- **地形管理**：进入该项目的所有地形数据管理页面，支持查看、编辑、文件切片、下载地形、删除地形文件等功能。
