# 首页

> 首页是平台的默认视图，提供项目概览、地图快速定位及主要状态监控功能。

![](/docs/public//images/三维页面介绍.png)

## 1. 搜索栏

- **项目搜索框**：输入项目名称或关键字，快速筛选并定位到目标项目。
- **重置视角**：将地图视角恢复到初始默认位置和缩放级别，便于重新浏览。

## 2. 地图区域

- **地图界面**：展示项目及子项目的地理位置分布，支持缩放、拖拽、点击图标查看项目详情。
- **项目地址图标**：显示当前选中项目的详细信息，例如状态、创建时间、未结项子项目等。

## 3. 项目状态

- **未结项**：统计并展示尚未结项的项目数量，点击可快速筛选。
- **结项**：统计并展示已结项的项目数量，点击可快速筛选。
- **图表分析**：点击进入，可查看项目状态。

## 4. 悬浮窗

- **离线预警**：显示设备离线导致的预警，点击可查看详情。
- **图像曲线预警**：展示基于图像或曲线触发的预警信息。
- **数据处理**：显示当前正在处理的数据任务状态（如地形融合、切片进度）。
- **监测点/面计算**：展示正在进行的监测点或监测面计算任务进度。
