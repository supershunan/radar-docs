# 三维页面

> 三维页面是平台的监测核心，提供三维地形可视化、监测数据叠加、形变分析等专业功能，支持从任意角度查看监测项目。

![](/../vpstatic/images/三维页面介绍.png)

## 1. 子项目信息

- **子项目名称**：显示当前选中的子项目名称。
- **最新数据时间**：展示当前子项目最新的监测数据的采集时间。
- **地图名称**：当前加载的地形数据名称，通常为上传的TIFF文件。
- **拍摄日期**：地形数据或影像的拍摄日期。
- **地图数据类型**：当前地形数据的格式类型，如TIFF、OSGB、DXF等。
- **位移视图-查看模式**：切换形变数据的显示方式，共六种，如位移视图-图像预警模式、速度视图-曲线预警模式等，可在视图功能界面进行修改。

## 2. 地图相关

- **地图**：显示当前项目的所处位置。
- **监测点/面**：用户手动添加的监测点/面，点击监测点/面的图标可查看形变曲线。
- **雷达图标**：显示雷达设备在地图上的位置。

## 3. 显示设置

- **设置**：对图例进行配置，如增删颜色条、图例复制等功能。
- **颜色条**：不同的颜色条区间反映不同的形变值。

## 4. 标记信息

- **编号**：当前选中监测点或监测面的唯一标识编号。
- **备注**：用户为监测点或监测面添加的自定义备注信息，便于识别和管理。

## 5. 工具栏

- **图层**：打开图层管理面板，可以切换图像类别、地图类别、地形类别以及控制子项目的不同图层的显隐。
- **视图**：支持切换三维场景的视图模式，对历史数据进行查询等功能。
- **工具**：用户在此页面可以手动添加监测点、监测面等，同时可以使用测量工具和关联分析功能。
- **报告**：快速生成当前子项目的监测报告或监测数据，支持预览和导出（时间间隔可按小时、天、月）。
- **系统**：管理监测数据（例如对监测点/面进行编辑、删除或定位）、视角书签和天空底色。

## 6. 坐标信息

- **纬度**：显示当前鼠标位置选中点的纬度坐标。
- **经度**：显示当前鼠标位置选中点的经度坐标。
- **高程**：显示当前鼠标位置选中点的高程值。

## 7. 视图导航

- **旋转地图**：按住并拖拽可自由旋转三维场景，从不同角度观察监测项目。
- **放大**：点击可放大地图视图，更清晰地查看局部细节。
- **房子**：点击后快速恢复到默认视角和位置，便于重新定位。
- **缩小**：点击可缩小地图视图，查看更大范围的场景。
