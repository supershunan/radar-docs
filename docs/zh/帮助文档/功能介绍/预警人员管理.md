# 预警人员管理

> 预警人员管理页面用于维护接收预警通知的联系人信息，支持对人员进行增删改查以及测试通知功能。

![](/../vpstatic/images/预警人员管理介绍.png)

## 1. 预警人员搜索栏

- **名称搜索**：输入姓名或部分姓名，快速查找目标预警人员。
- **查询**：点击后根据输入的关键词筛选预警人员列表。
- **重置**：清空搜索条件，恢复显示全部预警人员。

## 2. 预警人员基本信息

- **姓名**：预警人员的姓名。
- **邮件**：预警人员的电子邮箱，用于接收邮件通知。
- **手机号**：预警人员的手机号码，用于接收短信通知。

## 3. 编辑预警人员

- **新增**：添加新的预警人员，填写姓名、邮件、手机号信息。
- **邮箱测试**：向该人员的邮箱发送测试邮件，验证邮件通知是否可达。
- **短信测试**：向该人员的手机发送测试短信，验证短信通知是否可达。
- **编辑**：修改当前预警人员的姓名、邮件、手机号信息。
- **删除**：从系统中移除该预警人员。
